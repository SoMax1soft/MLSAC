## 🌐 Official Repository

This repository contains the official source code for the **MLSAC.NET** anti-cheat project.  
All official updates, releases, and issue tracking are maintained here.

---

## ✨ Features

- ⚡ **High performance** - minimal impact on server TPS
- 🔍 **Low false positives** - accurate and reliable checks
- 🧠 **Intelligent detection**
- 🔧 **Highly configurable**

---

## 🖥️ Supported Platforms

- Spigot
- Paper (Or any based Paper core)
- Java 17+
